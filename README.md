Android-TwoColorBalls
====
基于Android系统的双色球开奖信息,自动兑奖APP,项目通过抓取百度彩票数据获取双色球开奖结果以及历史开奖号码.用户可以将已买到最新一期的开奖号码记录到APP中,当最新一期开奖时APP会自动兑奖并且通知用户兑奖结果.<br>
